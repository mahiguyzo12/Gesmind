================================================================================
TROUBLESHOOTING
================================================================================

If SDK installation seems stuck:
1. Check available disk space: df -h
2. Check network: ping google.com
3. Wait longer (can take 10+ minutes on slow connections)
4. Check ~/android-sdk size: du -sh ~/android-sdk

If build fails after SDK completes:
1. Verify ANDROID_HOME is set correctly
2. Verify Java 21: java -version
3. Check Keystore password: gesmind2026
4. Delete build folder: rm -rf android/app/build/
5. Retry: ./build-apk-release.sh

If APK is too large:
1. Enable proguard/R8 in build.gradle
2. Remove unused dependencies
3. Use minifyReleaseCode in gradle properties
4. Check AndroidManifest for unused permissions
