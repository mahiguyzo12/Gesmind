ÉTAPE 3 : APK RELEASE BUILD (EN ATTENTE - CONFIGURATION REQUISE)
================================================================================

Status: ANDROID SDK SETUP REQUIRED

Pour générer l'APK release, vous devez :

Option A: Installer Android SDK (Recommandé)
──────────────────────────────────────────
1. Télécharger Android Studio depuis: https://developer.android.com/studio
2. Installer Android SDK Platform version 33 ou supérieure
3. Configurer ANDROID_HOME:
   export ANDROID_HOME=~/Library/Android/Sdk  # macOS/Linux
   export ANDROID_HOME=%APPDATA%\Local\Android\Sdk  # Windows

4. Exécuter:
   cd /workspaces/Gesmind/android
   ./gradlew assembleRelease

Output APK:
   android/app/build/outputs/apk/release/app-release.apk


Option B: Utiliser Termux (Android)
───────────────────────────────────
1. Installer Termux depuis F-Droid ou Play Store
2. Dans Termux, installer les outils:
   pkg install nodejs openjdk-17 android-tools
3. Cloner le dépôt et lancer:
   npm run build
   npx cap sync android
   cd android && ./gradlew assembleRelease

4. Récupérer l'APK généré:
   app/build/outputs/apk/release/app-release.apk


Option C: Utiliser un CI/CD (GitHub Actions)
────────────────────────────────────────────
Créer un fichier .github/workflows/android-build.yml pour générer 
automatiquement l'APK à chaque commit.
